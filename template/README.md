# Discord Server Template

This folder is for your exported Discord server template. Once your server is configured using the guides in this repo, you can publish it so others can create a copy with your channels, roles, and settings.

## How to Create a Template

1. Finish setting up your server using:
   - [channels/CHANNELS.md](../channels/CHANNELS.md)
   - [roles/ROLES.md](../roles/ROLES.md)
   - [rules/RULES.md](../rules/RULES.md)
2. Open **Server Settings → Server Template**.
3. Click **Generate Template**.
4. Name it (e.g. `Community Template`) and add a short description.
5. Copy the template link Discord provides.

## What Gets Included

Discord server templates typically include:

- Channel and category structure
- Role names, colors, and permissions
- Default channel permission overrides

Templates do **not** include:

- Message history or pinned messages
- Bot configurations or integrations
- AutoMod rules (configure manually from [automod/AUTOMOD.md](../automod/AUTOMOD.md))
- Member list or bans

Replace `YOUR_TEMPLATE_CODE` with the code from Discord.

## Importing a Template

If a template link is provided above:

1. Click the `discord.new` link (or open it in a browser).
2. Choose **Create a new server** or merge into an existing server.
3. After import, add bots per [bots/BOTS.md](../bots/BOTS.md) and configure AutoMod.
4. Customize rules text for your community.

## Notes

- Re-generate the template after major channel or role changes.
- Test the import flow with an alt account before sharing publicly.
- Keep sensitive or staff-only channels out of the template if you prefer to add them manually post-import.
